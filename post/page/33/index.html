Streaming data to InfluxDB from any bash command

InfluxDB is a great time series database, that’s recently been rebranded as part of the “TICK” stack, including data collectors, visualisation, and ETL/Alerting. I’ve yet to really look at the other components, but InfluxDB alone works just great with my favourite visualisation/analysis tool for time series metrics, Grafana. Getting data into InfluxDB is easy, with many tools supporting the native InfluxDB line input protocol, and those that don’t often supporting the carbon protocol (from Graphite), which InfluxDB also supports (along with others).

Sourcecode markup tweaks in Wordpress

I noticed in Ed Stevens’ blog posting here that some sourcecode he’d posted had certain lines highlighted. Wordpress provides the sourcecode tag for marking up sourcecode in blog posts. For example: cd /some/random/folder ls -l # do not run this next line! is much better presented as: [sourcecode language=“bash”] cd /some/random/folder ls -l # do not run this next line! [/sourcecode] by wrapping it in [sourcecode] tags I’ve known about the language=‘xx’ attribute that you can use with the tag, but Ed’s posting prompted me to check on the syntax and it turns out there a few tweaks one can use.
